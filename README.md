# AIがESを作成するアプリ<br><sub>ESや、会社の質問、ガクチカなどを自動で書くことができます。</sub>

rinnaの日本語のGPT-2を、就活サイトのESを用いてファインチューニングして、ESを書けるようにしました。
訓練にはあらゆる分野から140,000件のESを利用しました。

下の文章はAIが生成した文章です。
